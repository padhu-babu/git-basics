# Git From Zero to Hero 🚀

A fun, beginner-friendly guide to learning Git and trunk-based development — inspired by the Head First book series.

## Who Is This For?

**Absolute beginners.** If you've never used Git, never heard of version control, or just want a fresh start with a guide that doesn't put you to sleep — this is for you.

## Prerequisites

- A computer with a terminal (we'll show you how to open it)
- A text editor ([VS Code](https://code.visualstudio.com/) recommended)
- Zero Git knowledge required
