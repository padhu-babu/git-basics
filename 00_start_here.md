# 🚀 Git From Zero to Hero — A Beginner's Adventure

> *"The only way to learn Git is to use Git."*
> — Every developer who ever lived, probably.

Welcome, brave soul! You're about to learn **Git** — the tool that every developer on the planet uses, and the one thing standing between you and "I accidentally deleted everything and now I'm crying."

This guide assumes you know **absolutely nothing** about Git. No judgment. We all start somewhere. By the end, you'll be branching, merging, and pushing like you've been doing it for years.

**What you need before we start:**
- A computer (shocking, right?)
- A terminal / command line (we'll tell you how to open it)
- A text editor (VS Code, Notepad++, Sublime — anything works)
- A sense of adventure 🧭

Let's go! 🎬
